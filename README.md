# Student Record System

This is a simple C++ program that allows a user to input and display student records.

## 📌 Features
- Input student name and ID
- Store student data using structures
- Display all student records

## 🛠️ Technologies Used
- C++

## 🎯 Purpose
This project was created as part of my learning journey in programming. It demonstrates basic concepts such as:
- Structures
- Loops
- User input and output

## 🚀 How to Run
1. Compile the program using a C++ compiler
2. Run the executable file
3. Enter student details when prompted
